    Projects

    AI-Powered Gait Irregularity Detection

    Machine learning pipeline analyzing PhysioNet force-plate gait data to detect mobility impairments through biomechanical pattern recognition.


    • Built an end-to-end ML pipeline using scikit-learn for feature extraction, normalization, and cross-validation.
    • Engineered biomechanical features such as peak force, variability, and left–right symmetry indices.
    • Applied classification models to distinguish healthy vs. impaired gait cycles with stratified sampling.
    • Visualized gait cycle averages and confusion matrices using Matplotlib for interpretability.
    • Explored extensions with 1D CNN and LSTM architectures to capture temporal gait dynamics.

    Tech: Python | scikit-learn | NumPy | Pandas | Matplotlib

    Synexa - Healthcare Appointment Management Dashboard

    A full-stack healthcare management system that streamlines clinic workflows by enabling real-time appointment scheduling, cancellation tracking, and doctor availability visualization. The admin dashboard provides automated metrics for active, pending, and cancelled appointments, improving clinic decision-making and operational transparency.


    • Built with Next.js and Tailwind CSS for a modern, responsive UI
    • Integrated MongoDB for secure data storage and dynamic status updates
    • Designed API endpoints for patient, doctor, and appointment management
    • Implemented session control and form validation with NextAuth and React Hook Form
    • Supports analytics-ready exports for data visualization in Power BI

    Tech: Next.js | Tailwind CSS | MongoDB | NextAuth | React Hook Form

    IoMT Device Simulator - Real-Time Health Data Stream

    A Python-based simulator that generates and transmits real-time health sensor data - including heart rate, temperature, and oxygen saturation - to mimic Internet of Medical Things (IoMT) device behavior. The project enables data analytics and visualization pipelines for healthcare monitoring and anomaly detection.


    • Simulates multiple patient wearable devices streaming live data via HTTP and TCP sockets
    • Designed analytics dashboard with Streamlit to visualize time-series health metrics
    • Integrated alert system for threshold breaches and real-time event logging
    • Prepared data for ML-based classification and predictive health analytics

    Tech: Python | Flask | Streamlit | SQLite | Pandas | Matplotlib
